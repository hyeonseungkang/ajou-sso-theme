Initial of this design originally started from [this gist](https://gist.github.com/hyeonseungkang/ffaea0f43170f25cfae58f2f8c0bca4f). This gist created during my before discharge vacation in army, since univ. login sso page look ugly, also I was so boring.

개인적으로 본인의 CSS 역량 함양과 당시 유행한다던 뉴모피즘 디자인을 사용해보았기에 좋다고 평가한다.

In my opinion, this was good to raise skill in css and using neumorphism design popular at that time.

로그인 박스 주변으로 그림자를 주어 마치 자동차 유리 모서리처럼 디자인했다. 로그인박스와 배경에 음영차를 두는 디자인은 연세대 sso 페이지를 차용했다.

Made shadow around login box so it looks like corner of car window. Design from Yonsei Univ. sso page that has darkness difference between login box and background.
